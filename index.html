Geotag Photos using gpx file

This site allows you to add gps coordinates to jpeg photos taken in DSLR cameras (or converted from raw files).

Each photo is examined for it's date and time (TimeStamp) and this is compared to all points in the gpx file.
The point that has the smallest difference from the timestamp of the photo is chosen, and it's coordinates are added to the photo.
(Photos that were taken more than 15 minutes before the first point or after the last point in the gpx file are excluded)
In the end of the process, a zipped archive of all the photos that have been edited succesfully to add gps coordinates is downloaded.
You can track your route using an app like PixTrack (for iPhone) and export gpx file (that's free also!)

The pros of geotagging locally in JavaScript:

1. No need of internet access (once loading the site)
2. Process is completed quickly as no need to upload files to a remote server
3. Privacy is maintained as all processing occurs locally in browser's JavaScript
4. No need to install depencencies
5. Available from any operating system and browser
6. Easy to use and 7. Free forever!
